# 餐厅管理

## 新餐厅入驻流程

1.  **提交申请**: 商家通过入驻门户提交申请资料，包括营业执照、食品经营许可证、店铺照片等。
2.  **资质审核**: 平台运营团队审核商家提交的资质是否齐全、真实、有效。
3.  **信息配置**: 审核通过后，管理员协助商家在后台录入餐厅基本信息、地址、营业时间等。
4.  **菜单上传**: 商家上传商品信息，包括菜品名称、描述、价格、图片。管理员需要审核菜单内容是否合规。
5.  **上线培训**: 为商家提供后台系统操作培训。
6.  **正式上线**: 完成以上步骤后，将餐厅状态设置为“营业中”，使其在App中可见。

## 更新菜单和价格

- **商家自主更新**: 商家可以在其后台随时更新菜单，包括上架/下架商品、修改价格和库存。
- **平台审核**: 重要的信息变更（如大幅价格调整）可能需要平台审核，以防止价格欺诈。
- **批量更新**: 对于连锁品牌，平台支持批量更新多家分店的菜单。

## 处理商家违规行为

- **违规类型**:
    - 使用劣质食材。
    - 虚假宣传，图片与实物严重不符。
    - 私下引导用户进行线下交易。
    - 服务态度恶劣，频繁收到差评。
- **处理方式**:
    - **警告**: 对初次或轻微违规的商家进行口头或书面警告。
    - **流量限制**: 降低商家在平台上的曝光度。
    - **临时下线**: 要求商家停业整顿，直到问题解决。
    - **永久下线**: 对严重或屡教不改的商家，终止合作，永久下线。
